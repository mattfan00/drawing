I used a stock API provided by Alpha Vantage link for this project. This website provides a lot of useful statistics on stocks, such as the prices of a certain stock every 5 minutes (making it really great for data visualizations) or the stock's annual and quarterly earnings, but for my purposes I used an endpoint that just returns the opening and closing prices for a given stock every day. Using this data, I calculated the percentage changed from the previous closing price to the current closing price for GameStop, and visualized this percentage change through how far the rocket ship moves towards the moon. If the percent change is lower than 0%, the rocket ship blows up.
